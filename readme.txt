Name: Display Usergroup Moderator Usernames
Description: Display the usernames of usergroup moderators instead of the usergroup name
Website: https://github.com/MattRogowski/Display-Usergroup-Moderator-Usernames
Author: Matt Rogowski
Authorsite: https://matt.rogow.ski
Version: 1.0.0
Compatibility: 1.8.x
Files: 1

Information:
This plugin will list the names of users who have been added as usergroup moderators to a forum, rather than listing the names of the usergroups.

To Install:
Upload ./inc/plugins/usergroupmodusernames.php to ./inc/plugins/
Go to ACP > Plugins > Activate

Change Log:
